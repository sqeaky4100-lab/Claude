# Analysis Templates

This directory contains templates for common Windows environment analysis tasks.

## Available Templates

- Security audit checklists
- Group Policy review templates
- User account review templates
- Privilege escalation analysis
- Compliance check templates

## Using Templates

1. Copy a template file
2. Fill in your environment-specific information
3. Use the template as a guide for analysis
4. Ask Claude to review against the template

## What to Ask Claude

- "Use this template to analyze my GPO reports"
- "Check my environment against this security checklist"
- "Review user accounts using this template"
